# Incremental Backup in DataVault

## Overview
Only backup files that have changed since the last backup, saving time and bandwidth. This is enabled by default but can be configured in Backup Settings.

## How to Use
1. Open DataVault
2. Navigate to the Incremental Backup section
3. Follow the in-app instructions

## Tips and Best Practices
- Use this feature regularly to get the most value
- Explore all available options in settings
- Check our video tutorials for detailed walkthroughs
