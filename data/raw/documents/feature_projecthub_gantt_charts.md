# Gantt Charts in ProjectHub

## Overview
View project timelines and dependencies. Automatically updates when task dates change. Enable in Project Settings > Views.

## How to Use
1. Open ProjectHub
2. Navigate to the Gantt Charts section
3. Follow the in-app instructions

## Tips and Best Practices
- Use this feature regularly to get the most value
- Explore all available options in settings
- Check our video tutorials for detailed walkthroughs
