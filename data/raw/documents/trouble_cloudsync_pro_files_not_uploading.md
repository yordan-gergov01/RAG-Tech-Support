# Files not uploading in CloudSync Pro

## Problem Description
Files not uploading

## Solution
Verify you have sufficient storage space in your account. Check if the file size exceeds the 5GB limit per file. Also ensure the file format is supported.

## Additional Tips
- Make sure you're running the latest version of CloudSync Pro
- Check system requirements are met
- Try restarting the application
- Contact support if the issue persists
