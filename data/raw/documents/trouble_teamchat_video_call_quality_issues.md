# Video call quality issues in TeamChat

## Problem Description
Video call quality issues

## Solution
Ensure you have a stable internet connection (minimum 5 Mbps recommended). Close other bandwidth-heavy applications. Try switching from Wi-Fi to ethernet if possible.

## Additional Tips
- Make sure you're running the latest version of TeamChat
- Check system requirements are met
- Try restarting the application
- Contact support if the issue persists
